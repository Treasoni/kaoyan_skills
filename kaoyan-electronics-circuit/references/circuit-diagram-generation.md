# 822电路图生成规范

## 输出策略

生成电路图时优先使用 Obsidian Excalidraw `.md`，因为用户需要在 Obsidian 中继续修改、补标注、复习。只有当用户明确要求“教材级”“PDF”“LaTeX”时，才使用 `circuitikz` 或 SVG。

知识库说明文件：`考研专业课/📐 822电路图绘制规范.md`

---

## 工作流

1. 将题目或用户描述转成结构化网表：
   - 节点
   - 器件
   - 方向
   - 参数
   - 输入输出
2. 判断电路类型：
   - 共射、共集、共基
   - 差分、运放、负反馈
   - 功放、稳压、波形产生
   - 逻辑门、组合逻辑、时序逻辑
3. 选择模板或搭建拓扑。
4. 按网格放置器件，避免斜线和交叉。
5. 标注节点、电流方向、电压极性。
6. 如果是教学图，加入分析图层：
   - 直流通路
   - 交流通路
   - 微变等效
   - 反馈路径
7. 检查后保存。

---

## 器件符号要求

| 类型 | 要求 |
| --- | --- |
| 电阻 | 水平或竖直，标注 $R_B$、$R_C$、$R_E$、$R_L$ 等 |
| 电容 | 区分耦合电容和旁路电容，标注 $C_1$、$C_2$、$C_E$ |
| 电源 | 顶部 $V_{CC}$ 或 $V_{DD}$，底部 GND |
| NPN | 集电极在上，发射极在下，箭头向外 |
| PNP | 发射极箭头向内，注意电源极性 |
| MOSFET | 漏极在上，源极在下，标注 $U_{GS}$ |
| 运放 | 左侧输入，右侧输出，必须标 `+` 和 `-` |
| 反馈网络 | 用虚线或紫色线标注反馈路径 |

---

## 模电模板要求

### 共射放大电路

必须标注：
- $V_{CC}$
- $R_C$、$R_B$ 或 $R_{B1}$、$R_{B2}$
- $R_E$、$C_E$（若有）
- $C_1$、$C_2$
- $R_L$
- $u_i$、$u_o$
- $I_{BQ}$、$I_{CQ}$、$U_{CEQ}$

### 微变等效电路

必须标注：
- 交流地
- $r_{be}$
- 受控源
- $R_C$、$R_L$、$R_E$ 的等效关系
- $R_i$、$R_o$ 的观察端

### 运放电路

必须标注：
- 同相端和反相端
- 输入网络
- 反馈网络
- 输出端
- 工作区判断：只有线性区才能使用虚短虚断

---

## 检查清单

- 输入在左、输出在右、电源在上、地在下。
- 器件没有悬空节点。
- 三极管箭头方向正确。
- 运放同相端和反相端没有反。
- 直流通路中电容开路。
- 交流通路中电源交流接地。
- 反馈路径能看出取样和比较方式。
- 标注使用康华光符号体系。
