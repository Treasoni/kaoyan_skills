# 822知识结构与章节复盘

## 角色定位

本模块负责回答“现在学什么、这一章考什么、学到什么程度算过关”。它不平均展开教材，而是按 822 考研题型组织知识点。

总控文件：[[考研专业课/📘 822电子技术基础考研学习体系]]

---

## 路由场景

| 场景 | 输出重点 |
| --- | --- |
| 学新章节 | 学习目标、考法、前置知识、最低掌握标准 |
| 复习某章 | 高频题型、错题入口、回忆清单 |
| 查询知识结构 | 模块关系、前后置链路、SOP映射 |
| 启动模电 | 按大题优先，不按教材平均推进 |
| 数电回收 | 从章节笔记转向题型训练和错题复盘 |

## 822模块地图

### 数字电子技术

| 优先级 | 模块 | 学习重心 |
| --- | --- | --- |
| S | 时序逻辑电路 | 分析、设计、状态转换、自启动 |

| 优先级 | 模块 | 学习重心 |

## 章节卡片输出格式

每次生成章节复盘时，固定输出：

1. 本章考研定位。
2. 高频题型。
3. 前置知识。
4. 必背公式/表格/状态关系。
5. 易错点。
6. 对应SOP。
7. 最低验收标准。
8. 建议写入文件。

---

## 启动模电原则

- 先建立可做题的最小知识链：器件工作区 -> 共射放大 -> 运放 -> 反馈。
- 半导体器件不深挖材料物理，只保留判断工作区和等效模型所需内容。
- 每学一个电路，必须回答四问：工作区是什么、静态怎么算、动态怎么算、考试怎么设坑。
