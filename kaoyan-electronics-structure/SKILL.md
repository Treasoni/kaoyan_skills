# 822电子技术基础 - 知识点结构

## 技能概述

本技能提供822电子技术基础的知识点结构和关联管理：

1. **知识点图谱**：模电（8个模块）+ 数电（9个模块）
2. **前置知识关联**：提示学习某知识点前需要掌握的知识
3. **跨章节提示**：主动提示知识点之间的关联
4. **知识点卡片模板**：标准化笔记格式
5. **常见错误模式**：分类整理易错点

**设计原则**：
- 主动提示前置知识，避免知识断层
- 建立跨章节关联，形成知识网络
- 提供标准化的知识点卡片模板

## 考试重点总结

### 模电必考

1. **反馈类型判断** - 极高频率，难度高
   - 掌握短路法和瞬时极性法

2. **静态工作点计算** - 极高频率，难度中
   - 不要忘记减去$U_{BE}$

3. **差分放大电路** - 高频率，难度中
   - 注意四种接法的区别

4. **运算电路** - 高频率，难度中
   - 虚短虚断是关键

### 数电必考

1. **时序逻辑分析** - 极高频率，难度高
   - 驱动方程→状态方程→状态图→自启动

2. **计数器设计** - 极高频率，难度中
   - 置数法和复位法

3. **组合逻辑设计** - 高频率，难度中
   - 抽象→真值表→化简→实现

4. **卡诺图化简** - 高频率，难度中
   - 注意圈组原则和无关项

## 常见错误模式

### 模电错误

| 错误类型 | 例子 | 纠正方法 |
|----------|------|----------|
| 反馈类型判断错误 | 混淆电压/电流反馈 | 使用短路输出端法 |
| 忘记$U_{BE}$ | $I_B = V_{CC}/R_b$ | $I_B = (V_{CC} - U_{BE})/R_b$ |
| 忘记负载电阻 | $A_u = -\beta R_c/r_{be}$ | $R'_L = R_c // R_L$ |
| 混淆虚短虚断条件 | 比较器也用虚短 | 线性区才有虚短虚断 |

### 数电错误

| 错误类型 | 例子 | 纠正方法 |
|----------|------|----------|
| 忘记检查自启动 | 设计时序逻辑 | 无效状态必须能进入有效循环 |
| 卡诺图圈组错误 | 不按$2^n$圈组 | 按规则圈组，每个圈尽可能大 |
| 忘记异步置数端 | 忽略优先级 | 异步置数优先级最高 |
| 状态编码错误 | 编码不合理 | 满足$2^{n-1} < N \leq 2^n$ |

## 进度追踪规范

### 自动进度更新

**触发时机**：当生成新知识点笔记时

**执行步骤**：
1. 检查是否存在对应的章节进度文件（如 `1-数字与码制/📊 学习进度.md`）
2. 如不存在，创建进度文件
3. 如存在，更新对应知识点的状态为 `✅ 已完成`

**进度文件路径**：
```
考研专业课/数字电子技术/{章节}/📊 学习进度.md
考研专业课/模拟电子技术/{章节}/📊 学习进度.md
```
