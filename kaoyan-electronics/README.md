## 功能概览

`kaoyan-electronics` 是专为湖南大学822电子技术基础考研设计的学习技能，提供：

- **电路图智能解析** - 支持上传电路图截图，自动识别元件和拓扑结构
- **典型题型SOP库** - 17个考点的标准化解题流程
- **知识图谱主动关联** - 智能提示前置知识和跨章节关联
- **MemOS记忆追踪** - 跨设备同步的学习进度和错误记录
- **康华光符号体系** - 严格遵循指定教材符号，避免考场写错
- **Mermaid波形图** - 支持时序图、状态图在Obsidian中渲染
- **数学前置检查** - 学习前自动检查数学基础是否扎实
- **跨学科提醒** - 数学↔电子技术知识关联提醒
- **统一错误模型** - 学科标签支持跨技能聚合
- **调度信号处理** - 接收kaoyan-plan的调度信号

## 考试范围

### 模电部分（约50%）

1. 基本电子器件 - 二极管、稳压管、场效应管、BJT特性
2. 基本放大器 - 静态工作点、$A_u$、$R_i$、$R_o$、频率响应
3. 差分放大器 - $A_{ud}$、$A_{uc}$、$K_{CMRR}$
4. 功率放大器 - 效率$\eta$、$P_{om}$、交越失真
5. 集成运放 - 比例、加法、减法、有源滤波、电压比较
6. 反馈放大器 - 反馈类型判断、深度负反馈增益计算
7. 波形产生电路 - 正弦波、非正弦波产生
8. 线性稳压电源 - 构成、功能分析、参数计算

### 数电部分（约50%）

1. 数制与代码 - 二进制、十进制、十六进制转换
2. 逻辑运算 - 代数法、卡诺图化简
3. 基本逻辑单元 - 逻辑门、触发器（RS/JK/D/T/T'）
4. 组合逻辑单元 - 编码器、译码器、选择器、比较器、运算器
5. 时序逻辑电路 - 计数器、移位寄存器
6. 逻辑电路设计 - 组合逻辑分析设计、同步时序分析设计
7. 存储器 - 分类、特性、容量扩展
8. 脉冲电路 - 单稳态、多谐振荡器、施密特触发器
9. A/D D/A转换 - 分类、原理、参数、应用

## 使用方法

### 电路图分析

```
用户: [上传电路图截图] 帮我分析这个电路

AI将:
1. 识别电路结构和元件参数
2. 选择对应SOP模板
3. 生成静态分析、动态分析
4. 输出LaTeX格式笔记（康华光符号体系）
```

### 解题指导

```
用户: 这道题怎么做？[题目描述]

AI将:
1. 识别题目类型
2. 调用对应SOP
3. 按步骤引导解题
4. 提醒常见错误（如运放工作区判断）
```

### 知识点复习

```
用户: 帮我复习负反馈

AI将:
1. 显示知识点卡片
2. 提示前置知识
3. 显示跨章节关联
4. 提供练习题
5. 根据考试权重推荐复习时间
```

## 核心功能

### 1. 电路图智能解析 (kaoyan-electronics-circuit)

支持两种输入方式：
- 上传电路图截图
- 文字描述电路结构

处理流程：
```
电路图截图 → MCP工具识别 → 提取元件信息 → 分析电路拓扑 → 生成结构化笔记
```

### 2. 典型题型SOP库 (kaoyan-electronics-sop)

包含17个考点的标准化SOP模板：
- 模电SOP: 共射放大、负反馈判断、运算电路、差分放大、功率放大、波形产生、稳压电源
- 数电SOP: 逻辑化简、组合逻辑分析、时序逻辑分析、计数器设计、触发器应用等

### 3. 知识图谱主动关联 (kaoyan-electronics-structure)

主动提示：
- 前置知识点
- 跨章节关联
- 常见错误
- 练习建议

### 4. MemOS集成 (kaoyan-electronics-core)

- 跨设备同步学习进度
- 个性化错误追踪
- 智能复习提醒
- 优雅降级支持

### 5. 康华光教材符号体系

⚠️ 严格遵循康华光《电子技术基础》（第7版）符号体系：

| 物理量 | 康华光符号 | 其他教材 |
|--------|-----------|----------|
| 静态基极电流 | $I_{BQ}$ | $I_B$ |
| 静态集电极电流 | $I_{CQ}$ | $I_C$ |
| BJT输入电阻 | $r_{be}$ | $h_{ie}$ |
| 夹断电压 | $U_{GS(off)}$ | $U_P$ |

### 6. Mermaid波形图生成

支持在Obsidian中渲染时序图、状态图：

```mermaid
gantt
    title 计数器时序
    dateFormat X
    section 输出
    Q0     :0, 1, 0, 1, 0, 1
    Q1     :0, 0, 1, 1, 0, 0
```

### 7. 考点权重动态记忆

根据考试频率和重要性智能分配复习时间：

| 考点 | 权重 | 策略 |
|------|------|------|
| 反馈放大器 | 10/10 | 每天必练 |
| 时序逻辑电路 | 10/10 | 每天必练 |
| 存储器 | 2/3 | 考前突击 |

### 8. 运算电路工作区判断

防止误用虚短虚断：

| 电路类型 | 工作区 | 能用虚短？ |
|----------|--------|-----------|
| 反相比例放大 | 线性 | ✓ |
| 滞回比较器 | 非线性 | **✗** |

## 跨技能联合架构

### 数学前置检查

学习电子技术某些内容前，自动检查数学基础：

| 电子技术主题 | 必需数学基础 | 检查内容 |
|-------------|-------------|---------|
| 频率响应分析 | 复数运算、对数运算 | 能正确进行复数运算 |
| 暂态响应 | 微分方程、指数函数 | 能求解一阶线性微分方程 |
| 滤波器设计 | 拉普拉斯变换 | 理解s域分析 |

### 跨学科提醒示例

学习"频率响应"时自动提示：
```markdown
⚠️ 「频率响应」分析需要「复数运算」基础

**关键点**：
- 阻抗 Z = R + jX
- 幅值 |Z| = √(R²+X²)
- 相角 φ = arctan(X/R)

💡 建议先复习数学「复数运算」再深入学习频率响应
```

### 数学→电子技术知识图谱

```yaml
"复数运算" → ["频率响应分析", "交流电路", "滤波器设计"]
"微分方程" → ["暂态响应", "RC/RL电路", "一阶电路分析"]
"积分" → ["RC充放电", "能量计算", "电容储能"]
"拉普拉斯变换" → ["s域分析", "传递函数"]
```

    - "#mistake_record"
    - "#subject_electronics"
