## 错误记录数据模型

### 基本结构

```yaml
mistake_record:
  # 基本信息
  record_id: string
  timestamp: datetime
  user_id: string

  # 错误内容
  knowledge_point: string        # 知识点名称
  mistake_type: enum            # 错误类型
  original_understanding: string # 原始理解
  correction: string            # 正确理解

  # 分类标签
  module: enum                  # 模电/数电
  chapter: string               # 章节
  difficulty: int               # 难度1-5

  # 追踪信息
  mistake_count: int            # 错误次数
  last_occurrence: datetime     # 最后一次错误
  is_resolved: boolean          # 是否已解决
```

## 错误类型分类

### 模电错误类型

```yaml
# 概念类错误
concept_confusion:
  - 混淆电压放大与电流放大
  - 混淆串联反馈与并联反馈
  - 混淆甲类、乙类、甲乙类功放
  - 混淆虚短与虚断的条件

# 计算类错误
calculation_error:
  - 忘记单位换算（mA/A, kΩ/Ω）
  - 忽略负载电阻$R_L$
  - 错误计算并联电阻
  - 忘记$(1+\beta)$因子

# 电路误读
circuit_misread:
  - 误判电路组态
  - 忽略偏置电路
  - 漏看反馈支路
  - 错误识别元件类型

# 条件遗漏
forgot_condition:
  - 忘记检查放大区条件
  - 忽略深度负反馈条件
  - 忘记振荡条件
  - 忽略稳定性条件
  - 忘记判断运放工作区（虚短虚断适用条件）
  - 忘记检查反馈极性

# 公式误用
formula_misuse:
  - 错误使用$r_{be}$公式
  - 混淆$A_u$与$A_{ud}$
  - 错误应用反馈公式
  - 混淆瞬时值与有效值
```

### 数电错误类型

```yaml
# 概念类错误
concept_confusion:
  - 混淆组合逻辑与时序逻辑
  - 混淆同步与异步
  - 混淆电平触发与边沿触发
  - 混淆ROM与RAM

# 逻辑错误
logic_error:
  - 卡诺图圈组错误
  - 状态转换图画错
  - 忘记异步置数端
  - 逻辑函数化简错误

# 设计算法错误
design_error:
  - 状态编码错误
  - 驱动方程错误
  - 忘记检查自启动
  - 逻辑化简不彻底

# 时序错误
timing_error:
  - 忽略建立时间
  - 忽略保持时间
  - 忽略传播延迟
  - 时钟关系错误

# 真值表错误
truth_table_error:
  - 真值表填写错误
  - 状态表填写错误
  - 激励表错误
```

### 示例1: 负反馈类型判断错误

```markdown
## 错误记录：电压反馈 vs 电流反馈

**错误时间**: 2025-02-26
**错误类型**: concept_confusion
**错误次数**: 3

### 原始理解
我看输出端有没有直接连到反馈网络，有就是电压反馈。

### 正确理解
应该用"短路输出端"法判断：
- 短路输出端，反馈信号消失 → 电压反馈
- 短路输出端，反馈信号仍存在 → 电流反馈

### 混淆原因
- 没有掌握正确的判断方法
- 靠直觉而不是靠方法

### 区分方法
| 判断方法 | 电压反馈 | 电流反馈 |
|----------|----------|----------|
| **输出取样** | 取自输出端 | 取自非输出端 |
| **短路法** | 短路输出→反馈消失 | 短路输出→反馈仍存在 |
| **特征** | $X_f \propto U_o$ | $X_f \propto I_o$ |
| **稳定对象** | 稳定输出电压 | 稳定输出电流 |

### 记忆口诀
```
电压反馈看输出：短路负载反馈无
电流反馈看输出：短路负载反馈留
```

### 相关练习

2. 判断反馈类型：[电路图2]
3. 判断反馈类型：[电路图3]

### 状态: ⚠️ 未解决
```

### 示例2: 静态工作点计算错误

```markdown
## 错误记录：共射电路静态计算

**错误时间**: 2025-02-26
**错误类型**: calculation_error
**错误次数**: 2

### 题目
已知$V_{CC}=12V, R_b=300k\Omega, R_c=3k\Omega, \beta=50, U_{BE}=0.7V$，求静态工作点。

### 错误解答
$$
I_B = \frac{V_{CC}}{R_b} = \frac{12}{300k} = 40\mu A
$$
$$
I_C = \beta I_B = 50 \times 40\mu A = 2mA
$$
$$
U_{CE} = V_{CC} - I_C R_c = 12 - 2 \times 3 = 6V
$$

### 正确解答
$$
I_B = \frac{V_{CC} - U_{BE}}{R_b} = \frac{12 - 0.7}{300k} = 37.7\mu A
$$
$$
I_C = \beta I_B = 50 \times 37.7\mu A = 1.88mA
$$
$$
U_{CE} = V_{CC} - I_C R_c = 12 - 1.88 \times 3 = 6.36V
$$

### 错误原因
- 计算$I_B$时忘记减去$U_{BE}$
- 导致后续结果都有偏差

### 关键要点
1. $I_B$计算公式：$I_B = \frac{V_{CC} - U_{BE}}{R_b}$
2. $U_{BE}$通常取0.7V（硅管）
3. 先计算$I_B$，再计算$I_C$，最后计算$U_{CE}$

### 检查清单
- [ ] $I_B$计算是否减去$U_{BE}$
- [ ] 单位是否统一（mA, kΩ, V）
- [ ] $\beta$是否正确使用
- [ ] $U_{CE}$是否大于$U_{CES}$（约0.3V）

### 状态: ✓ 已解决
```

## 使用说明

1. **记录错误**：每次出现错误时，使用对应模板记录
2. **分析原因**：深入分析错误原因，而不是只看结果
3. **总结要点**：提炼关键要点和记忆口诀
4. **定期复习**：定期回顾错误记录
5. **更新状态**：问题解决后标记为"已解决"
