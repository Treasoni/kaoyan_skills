# 考研英语复习技能 (Kaoyan English)

> 专注于考研英语词汇复习的智能助手，支持MemOS记忆系统，实现词汇学习状态的持续追踪和跨设备同步。

## 版本: v3.1.0

## 功能概述

| 功能 | 说明 |
|------|------|
| **词汇整理** | 从墨墨/百词斩等APP导出的PDF中提取词汇，生成结构化笔记 |
| **真题语境** | 优先使用真题例句，将单词串联成真题风格的语境文章 |
| **僻义预警** | 自动检测熟词僻义（如 address 的"处理"义），标记陷阱词 |
| **复习计划** | 基于考研适配的 SM-2 算法，根据考试倒计时自动调整复习间隔 |
| **快速查词** | 查询单词音标、释义、搭配、同义词等 |
| **单词测试** | 生成词义、搭配、拼写等多种测试题 |
| **写作训练** | 写作替换升级、汉译英练习、词义辨析 |

## 核心特性

### 1. 真题语境优先

- 优先检索近20年真题例句
- 外刊同源补充（经济学人、卫报等）
- AI生成仅作为最后补充

### 2. 熟词僻义预警 ⚠️

自动标记考研陷阱词：

| 单词 | 常见义 | **考研僻义** | 频率 |
|------|--------|-------------|------|
| address | 地址 | **vt. 处理，解决** | 80% |
| school | 学校 | **n. 流派，学派** | 70% |
| novel | 新颖的 | **n. 长篇小说** | 65% |
| fine | 好的 | **n./v. 罚款** | 60% |

### 3. 考研适配的复习算法

根据考试倒计时自动调整复习间隔：

```
基础期（>100天）  → 标准SM-2算法
强化期（30-100天） → 高频词间隔-20%，僻义词-30%
冲刺期（7-30天）   → 强制缩短间隔
极限冲刺（<7天）    → 高频词最多3天一复习
```

### 4. MemOS记忆系统 (v3.0+)

- 💾 **词汇卡片持久化**: SM-2算法状态永久保存
- 📈 **学习进度追踪**: 记录每个单词的复习次数、正确率、遗忘率
- 🔗 **跨设备同步**: 支持多设备间词汇学习进度同步
- 🛡️ **优雅降级**: MemOS不可用时自动降级为无状态模式

### 5. 鲁棒性增强 (v3.1)

- 🔄 **画像刷新机制**: 30天未更新时触发确认询问
- 🚨 **词汇欠账熔断**: 超过200个待复习词时强制暂停新词
- 📝 **复习会话upsert**: 同一天多次复习自动归档历史版本
- 💙 **词汇疲劳追踪**: 连续3天疲劳时触发缓解模式

## 使用场景

```

生成考研英语复习文章       → 真题语境文章

```

## v3.1 新增功能

### 1. 画像刷新机制 🔄

**问题**: 英语水平是动态的，但记忆是静态的

**解决方案**: 超过30天未更新时自动触发确认询问

```markdown
# ⚠️ 用户画像确认

你的英语学习画像已**31天未更新**，请确认以下设置是否有变化：

## 当前设置
- **英语水平**: 中级
- **每日新词目标**: 50个
- **复习重点**: 均衡
- **僻义敏感度**: 中

## 请确认
1. 英语水平有变化吗？(基础/中级/高级)
2. 每日新词目标需要调整吗？
3. 复习重点需要调整吗？
```

### 2. 词汇欠账熔断 🚨

**问题**: 连续欠账会无限制累积

**解决方案**: 超过200个待复习词时强制暂停新词

```markdown
# ⚠️ 词汇紧急模式

你的待复习词汇已达**205个**，超过安全阈值（200个）

**紧急措施**：
- 🔴 暂停新词学习
- 🔴 今日专注复习
- 🔴 待复习词降至安全阈值后自动恢复正常
```

### 3. 词汇疲劳追踪 💙

**问题**: 词汇学习容易产生倦怠

**解决方案**: 连续3天反馈疲劳时触发缓解模式

```markdown
# 💙 词汇学习提醒

检测到你连续3天词汇学习疲劳，今日已自动调整为**词汇缓解模式**：

**调整措施**：
- 减少新词量50%
- 增加真题语境阅读
- 暂停僻义词训练
- 增加写作应用练习
```

```markdown
### 普通复习

```

## MemOS数据存储

本技能使用MemOS记忆系统存储以下数据：

| 数据类型 | 用途 | 保留策略 |
|----------|------|----------|
| 用户画像 | 英语水平、考试信息、学习偏好 | 永久保存 |
| 词汇卡片 | SM-2状态、复习历史、僻义预警 | 每词每用户1条 |
| 复习记录 | 每日复习会话、正确率统计 | 按需保留 |
| 测试记录 | 测试成绩、错误分析 | 永久保存 |
| 心理状态 | 词汇疲劳追踪 | 最近30天 |

**隐私说明**: 所有数据通过MemOS工具管理，遵循用户配置的数据存储策略。

## 数据结构

支持 Obsidian Flavored Markdown 格式，可配合以下工具使用：

- **Obsidian**: 管理词汇库和复习笔记
- **Anki**: 导入间隔重复卡片
- **PDF导出**: 打印便携词汇表

## 依赖技能

- `obsidian-markdown` - 创建笔记、管理词汇库
- `docx` - 导出Word文档、PDF打印

## 降级策略

当MemOS不可用时，技能自动降级为v2.0.0无状态模式：
- ✅ 所有核心功能正常工作
- ❌ 不保存学习历史到持久存储
- ❌ 不进行跨设备同步
- ❌ 不启用智能追踪（疲劳、欠账熔断等）

## 使用建议

### 基础期（>100天）
- 每日新词：50个
- 关注熟词僻义预警
- 建立词汇库基础

### 强化期（30-100天）
- 每日新词：30个
- 重点复习僻义词
- 使用真题语境文章阅读训练
- 开始写作替换练习

### 冲刺期（<30天）
- 每日新词：0-10个
- 重点突破僻义词和高频词
- 加强写作输出训练
