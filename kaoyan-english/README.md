# 考研英语复习技能 (Kaoyan English)

> 专注于考研英语词汇复习的智能助手，支持真题语境、熟词僻义预警、间隔重复记忆和写作训练。

## 功能概述

| 功能 | 说明 |
|------|------|
| **词汇整理** | 从墨墨/百词斩等APP导出的PDF中提取词汇，生成结构化笔记 |
| **真题语境** | 优先使用真题例句，将单词串联成真题风格的语境文章 |
| **僻义预警** | 自动检测熟词僻义（如 address 的"处理"义），标记陷阱词 |
| **复习计划** | 基于考研适配的 SM-2 算法，根据考试倒计时自动调整复习间隔 |
| **快速查词** | 查询单词音标、释义、搭配、同义词等 |
| **单词测试** | 生成词义、搭配、拼写等多种测试题 |
| **写作训练** | 写作替换升级、汉译英练习、词义辨析 |

## 使用场景

```
整理考研英语单词         → 词汇整理
生成考研英语复习文章       → 真题语境文章
间隔重复背单词            → 复习计划
查单词 (考研语境)         → 快速查词
考研英语单词测试          → 单词测试
熟词僻义 / 陷阱词         → 僻义预警
写作替换 / 汉译英         → 写作训练
```

## 核心特性

### 1. 真题语境优先

- 优先检索近20年真题例句
- 外刊同源补充（经济学人、卫报等）
- AI生成仅作为最后补充

### 2. 熟词僻义预警

自动标记考研陷阱词：

| 单词 | 常见义 | **考研僻义** | 频率 |
|------|--------|-------------|------|
| address | 地址 | **vt. 处理，解决** | 80% |
| school | 学校 | **n. 流派，学派** | 70% |
| novel | 新颖的 | **n. 长篇小说** | 65% |

### 3. 考研适配的复习算法

根据考试倒计时自动调整复习间隔：

```
基础期（>100天）  → 标准SM-2算法
强化期（30-100天） → 高频词间隔-20%，僻义词-30%
冲刺期（7-30天）   → 强制缩短间隔
极限冲刺（<7天）    → 高频词最多3天一复习
```

## 输出示例

### 单词卡片

```markdown
# address

## 基本信息
**音标**: /əˈdres/
**词性**: vt./n.

## ⚠️ 僻义预警 [critical]
> 此词在考研中 **80%** 考查僻义"处理"

**考研常考僻义**: vt. 处理；解决；着手处理

### 真题例句
> 2022年真题 完形填空
> The committee failed to **address** the concerns raised by the public.
```

### 每日复习计划

```markdown
## 今日复习 (2025-02-25)

### 高优先级（僻义词+高频词）
- [ ] address - 间隔3天 [僻义词critical]
- [ ] school - 间隔5天 [僻义词warning]

### 普通复习
- [ ] exemplify - 间隔7天
```

## 数据结构

支持 Obsidian Flavored Markdown 格式，可配合以下工具使用：

- **Obsidian**: 管理词汇库和复习笔记
- **Anki**: 导入间隔重复卡片
- **PDF导出**: 打印便携词汇表

## 依赖技能

- `obsidian-markdown` - 创建笔记、管理词汇库
- `docx` - 导出Word文档、PDF打印

## 使用建议

### 基础期（>100天）
- 每日新词：50个
- 关注熟词僻义预警
- 建立词汇库基础

### 强化期（30-100天）
- 每日新词：30个
- 重点复习僻义词
- 使用真题语境文章阅读训练
- 开始写作替换练习

### 冲刺期（<30天）
- 每日新词：0-10个
- 重点突破僻义词和高频词
- 加强写作输出训练
