## 智能查重机制

**按"名称 + 关键公式"去重，避免同一知识点被多次记录**

### 定义/定理查重规则

```
去重键 = 标准化名称 + 关键公式指纹

标准化名称：
  - "极限定义" → "极限"
  - "极限的ε-δ定义" → "极限"
  - "函数极限定义" → "极限"
  - 去除：的、之、函数、定理、定义 等修饰词
  - 统一：ε→e, δ→d, α→a, β→b

关键公式指纹：
  - 提取公式中的核心符号序列
  - 示例："$\lim_{x \to a} f(x) = L$" → "lim_x_to_a_f(x)_L"
```

### 个人理解查重规则

```
去重键 = 来源文件 + 理解标题
```

### 易错点查重规则

```
去重键 = 核心错误类型（标准化）
```

## 特殊处理

### LaTeX 公式

- 保持原格式，不做修改
- **LaTeX 闭合校验**：检查 `$$` 和 `$` 是否成对出现

### 表格中的绝对值符号

> **在生成 Markdown 表格时，绝对值符号必须使用 `\vert` 而非 `|`！**

| 场景 | 正确写法 | 错误写法 |
|:---|:---|:---|
| 表格内公式 | `$\vert f(x) \vert$` | `$|f(x)|$` ❌ |
| 普通段落 | `$|f(x)|$` | 可以使用 |

### 图片/附件处理

- **必须保留** `![[pasted image ...]]` 格式的图片链接
- 图片链接放在相关知识点附近

### 知识孤岛检测

```
for 每个笔记文件:
    if 该文件提取到任何核心内容:
        标记为 ✅ 正常
    else:
        记录到"孤岛列表"
```
