# 考研规划Skill (Kaoyan Plan)

> 智能考研学习计划生成器，**四技能联合架构中央调度器**，支持MemOS记忆系统集成，实现个性化学习追踪和跨设备同步。

## 版本: v3.2.0

## 核心特性

### 基础功能 (v2.0+)
- 🎯 **渐进式输入**: 极简/标准/高级三种模式，适应不同使用场景
- 📊 **课表解析**: 支持图片/PDF/口头描述格式的课表
- 🌙 **Chronotype适配**: 支持晨型人/夜型人个性化时段分配
- 😊 **疲劳度管理**: 基于主观感受的混合疲劳度模型
- 📅 **智能排课**: 基于考研通用规则自动分配学习任务

### 实战补丁 (v2.1+)
- ⚡ **任务欠账处理**: 自动检测未完成任务，生成补课方案
- 🔄 **周日复盘**: 每周日自动触发复盘模式，强制错题重做
- ⏱️ **最小块时长**: 数学分时段检查，防止碎片化学习

### MemOS集成 (v3.0+)
- 💾 **用户画像持久化**: 保存考试日期、目标、作息类型等个性化配置
- 📈 **学习进度追踪**: 记录每日计划执行情况，统计完成率和欠账时长
- 🔗 **跨设备同步**: 支持多设备间学习进度同步
- 🛡️ **优雅降级**: MemOS不可用时自动降级为无状态模式

### 鲁棒性增强 (v3.1+)
- 🔄 **画像自动刷新**: 超过30天未更新时自动触发画像确认询问
- 🚨 **欠账熔断保护**: 累计欠账超过10小时时强制暂停新内容
- 📝 **计划版本控制**: 同一天多次生成计划时自动归档历史版本
- 💙 **心理状态追踪**: 连续3天疲惫时自动触发心理调节模式

### 🆕 跨技能联合架构 (v3.2)
- 🧠 **中央调度**: 作为"大脑"协调 math/english/electronics 三个技能
- ⚖️ **全局权重计算**: 根据考试倒计时动态调整各科权重
- 🔗 **跨学科协调**: 数学↔专业课知识关联提醒
- 📊 **统一错误模型**: 全科错题记录统一格式
- 📋 **周日全科复盘**: 跨学科错题分析报告

## 快速开始

### 极简模式（推荐首次使用）

只需提供课表信息：

```
"我周一上午没课，下午有英语，今天想学高数"
```

即可生成基于默认考研规则的学习计划。首次使用后自动创建用户画像。

### 标准模式

提供考试日期：

```
"我12月21日考试，数学比较薄弱"
```

生成考虑倒计时的分阶段计划。考试日期和科目权重会保存到用户画像。

### 高级模式

提供完整数据：

```
"我是夜型人，晚上效率高。数学进度落后10h，这周学了18h英语12h专业课，感觉有点累"
```

生成基于数据分析的个性化调整方案。从MemOS读取历史学习记录进行精准分析。

## v3.1 新增功能

### 1. 画像自动刷新 🔄

**问题**: 用户的学习状态是动态的，但记忆是静态的

**解决方案**: 超过30天未更新时自动触发确认询问

```markdown
# ⚠️ 用户画像确认

你的学习画像已**31天未更新**，请确认以下设置是否有变化：

## 当前设置
- **作息类型**: 晨型人
- **疲劳敏感度**: 中
- **科目权重**: 数学1.5 | 英语1.2 | 专业课1.0 | 政治0.6

## 请确认
1. 作息类型有变化吗？
2. 科目优先级需要调整吗？
3. 疲劳敏感度有变化吗？
```

### 2. 欠账熔断保护 🚨

**问题**: 连续欠账会无限制累积，导致用户崩溃

**解决方案**: 超过10小时自动触发紧急恢复模式

```markdown
# ⚠️ 紧急恢复模式

你的累计欠账已达 **11小时**，超过安全阈值（10小时）

**紧急措施**：
- 🔴 暂停所有新内容学习
- 🔴 今日专注补账
- 🔴 欠账降至安全阈值后自动恢复正常
```

### 3. 计划版本控制 📝

**问题**: 一天多次生成计划会导致记忆冗余

**解决方案**: 使用upsert逻辑，自动归档历史版本

```
#daily_plan_current (1条): 最新版本
#daily_plan_history (多条): 历史版本归档
#date_YYYY-MM-DD: 所有版本索引
```

### 4. 心理调节模式 💙

**问题**: 连续学习容易导致倦怠

**解决方案**: 连续3天疲惫时触发心理调节模式

```markdown
# 💙 心力提醒

检测到你连续3天反馈疲惫，今日已自动调整为**心理调节模式**：

**调整措施**：
- 学习量减少30%
- 增加休息时间
- 任务以低难度复习为主
- 添加鼓励和正向反馈
```

## 实战补丁详解

### 补丁1: 任务欠账处理 ⚡

**问题**: 计划赶不上变化是考研常态

**欠账等级表**:

| 欠账时长 | 等级 | 处理策略 |
|----------|------|----------|
| < 1小时 | 轻微 | 今日碎片时间补完 |
| 1-3小时 | 中等 | 压缩今日低优先任务补课 |
| 3-10小时 | 严重 | 建议补课日 |
| > 10小时 | 🔴 **熔断** | **强制暂停新内容** |

### 补丁2: 周日复盘 🔄

**问题**: 考研需要反复复盘，不能一直学新内容

**包含内容**:
- 本周完成度统计
- 2小时错题重做（强制）
- 进度对齐检查
- 下周计划调整

### 补丁3: 最小块时长 ⏱️

**问题**: 数学需要大块时间（3小时+），单词可以碎片化

**最小时长要求**:

| 科目 | 最小时长 | 原因 |
|------|----------|------|
| 数学 | 1.5小时 | 需要进入状态 |
| 英语阅读 | 1小时 | 需要完整语境 |
| 专业课 | 1小时 | 需要深度思考 |
| 单词 | 15分钟 | 可碎片化 |
| 政治选择 | 20分钟 | 可碎片化 |

## 个性化功能

### Chronotype（作息类型）

**晨型人**（默认）：
- 上午：数学、高难度内容
- 晚上：轻松复习、单词

**夜型人**：
- 上午：单词、轻松内容
- 晚上：数学、高难度内容

### 疲劳度反馈

选择您当前的状态：

| 主观感受 | 调整措施 |
|----------|----------|
| 精力很好 ✨ | 正常计划，可适当增加挑战 |
| 正常 🙂 | 正常计划 |
| 有点累 😌 | 减少10-20%学习量，增加休息 |
| 很累 😫 | 减少30%学习量，建议休息日 |

## 核心规则

### 科目优先级

```
数学 > 英语 > 专业课 > 政治
```

### 🆕 动态权重分配 (v3.2)

根据考试倒计时自动调整各科权重：

| 阶段 | 天数 | 数学 | 英语 | 专业课 | 政治 |
|------|------|------|------|--------|------|
| 基础期 | >300 | 1.2 | 1.44 | 0.8 | 0.48 |
| 强化期 | 180-300 | 1.5 | 1.2 | 1.0 | 0.6 |
| 十月强化期 | 90-180 | 1.95 | 0.96 | 1.5 | 1.2 |
| 冲刺期 | 30-90 | 2.25 | 0.72 | 1.8 | 1.2 |
| 极限冲刺 | <30 | 3.0 | 0.48 | 2.0 | 1.2 |

### 时段适配（晨型人）

```
上午（08:00-12:00）：数学、英语单词
下午（14:00-18:00）：英语阅读、专业课
晚上（19:00-23:00）：专业课、政治、复盘
```

### 分阶段策略

```
基础期（>100天）：全面打基础，四科均衡
强化期（30-100天）：增加刷题量
冲刺期（7-30天）：真题模拟为主
极限冲刺期（<7天）：只复习，不学新内容
```

## MemOS数据存储

本技能使用MemOS记忆系统存储以下数据：

| 数据类型 | 用途 | 保留策略 |
|----------|------|----------|
| 用户画像 | 个性化配置 | 永久保存 |
| 每日计划 | 计划版本控制 | 每日1条current + 多条history |
| 学习进度 | 任务完成统计 | 按需保留 |
| 心理状态 | 疲劳追踪 | 最近30天 |
| 周日复盘 | 复盘数据 | 永久保存 |

**隐私说明**: 所有数据通过MemOS工具管理，遵循用户配置的数据存储策略。

## 🆕 跨技能调度 (v3.2)

作为中央调度器，kaoyan-plan 协调其他三个考研技能：

### 调度规则

| 条件 | 目标技能 | 动作 | 说明 |
|------|----------|------|------|
| 晨型人 + 高精力 | kaoyan-math | `high_difficulty_sop` | 高难度数学SOP |
| 疲劳检测 (fatigue > 0.6) | kaoyan-english | `vocabulary_review_mode` | 轻量词汇复习 |
| 数学欠账 > 3h | kaoyan-english | `memory_compression_mode` | 英语时间压缩 |
| 学习"频率响应" | kaoyan-math | `cross_subject_reminder` | 提示"复数运算" |
| 周日复盘 | 全部技能 | `weekly_error_analysis` | 全科错题汇总 |
