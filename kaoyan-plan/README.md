# 考研规划Skill (Kaoyan Plan)

快速开始使用考研规划技能 v2.1。

## 核心特性

- 🎯 **渐进式输入**: 极简/标准/高级三种模式，适应不同使用场景
- 📊 **课表解析**: 支持图片/PDF/口头描述格式的课表
- 🌙 **Chronotype适配**: 支持晨型人/夜型人个性化时段分配
- 😊 **疲劳度管理**: 基于主观感受的混合疲劳度模型
- 📅 **智能排课**: 基于考研通用规则自动分配学习任务
- ⚡ **任务欠账处理**: 自动检测未完成任务，生成补课方案
- 🔄 **周日复盘**: 每周日自动触发复盘模式，强制错题重做
- ⏱️ **最小块时长**: 数学分时段检查，防止碎片化学习

## 快速开始

### 极简模式（推荐首次使用）

只需提供课表信息：

```
"我周一上午没课，下午有英语，今天想学高数"
```

即可生成基于默认考研规则的学习计划。

### 标准模式

提供考试日期：

```
"我12月21日考试，数学比较薄弱"
```

生成考虑倒计时的分阶段计划。

### 高级模式

提供完整数据：

```
"我是夜型人，晚上效率高。数学进度落后10h，这周学了18h英语12h专业课，感觉有点累"
```

生成基于数据分析的个性化调整方案。

## 实战补丁 v2.1

### 补丁1: 任务欠账处理 ⚡

**问题**: 计划赶不上变化是考研常态

**解决方案**: 自动检测昨日未完成任务，生成补课方案

**示例**:
```
"昨天的高数练习没做完，今天怎么办？"

→ AI生成3个补课方案：
1. 压缩英语时间补高数
2. 延后到周末补
3. 今天专注补账
```

### 补丁2: 周日复盘 🔄

**问题**: 考研需要反复复盘，不能一直学新内容

**解决方案**: 每周日晚上自动触发复盘模式

**包含内容**:
- 本周完成度统计
- 2小时错题重做（强制）
- 进度对齐检查
- 下周计划调整

### 补丁3: 最小块时长 ⏱️

**问题**: 数学需要大块时间（3小时+），单词可以碎片化

**解决方案**: 检查时段是否满足科目最小时长要求

**最小时长要求**:

| 科目 | 最小时长 | 原因 |
|------|----------|------|
| 数学 | 1.5小时 | 需要进入状态 |
| 英语阅读 | 1小时 | 需要完整语境 |
| 专业课 | 1小时 | 需要深度思考 |
| 单词 | 15分钟 | 可碎片化 |
| 政治选择 | 20分钟 | 可碎片化 |

**示例**:
```
原计划：09:00-10:00 数学（1小时）
↓ 检测不满足1.5小时要求
自动替换为：09:00-10:00 单词复习
```

## 输出示例

### 任务欠账处理输出

```markdown
# ⚠️ 任务欠账处理

## 欠账信息
- 未完成任务：高数第五章练习
- 欠账时长：2小时
- 欠账等级：中等

## 建议处理方案

**方案1：压缩英语时间补高数**（推荐）
- 今日英语阅读时间减半
- 腾出1小时补高数
- 剩余1小时晚上补完

**方案2：延后到周末补**
- 今天按原计划进行
- 周末安排2小时补账

**方案3：今天专注补账**
- 暂停今天所有新内容
- 全天补完欠账+复习

请选择方案（1/2/3），或输入自定义方案：
```


## 个性化功能

### Chronotype（作息类型）

**晨型人**（默认）：
- 上午：数学、高难度内容
- 晚上：轻松复习、单词

**夜型人**：
- 上午：单词、轻松内容
- 晚上：数学、高难度内容

### 疲劳度反馈

选择您当前的状态：

| 主观感受 | 调整措施 |
|----------|----------|
| 精力很好 ✨ | 正常计划，可适当增加挑战 |
| 正常 🙂 | 正常计划 |
| 有点累 😌 | 减少10-20%学习量，增加休息 |
| 很累 😫 | 减少30%学习量，建议休息日 |

## 核心规则

### 科目优先级

```
数学 > 英语 > 专业课 > 政治

权重：
- 数学：1.5
- 英语：1.2
- 专业课：1.0
- 政治：0.6
```

### 时段适配（晨型人）

```
上午（08:00-12:00）：数学、英语单词
下午（14:00-18:00）：英语阅读、专业课
晚上（19:00-23:00）：专业课、政治、复盘
```

### 分阶段策略

```
基础期（>100天）：全面打基础，四科均衡
强化期（30-100天）：增加刷题量
冲刺期（7-30天）：真题模拟为主
极限冲刺期（<7天）：只复习，不学新内容
```

## 常用命令

```
"生成考研学习计划"     - 使用极简模式生成今日计划
"制定周计划"           - 生成本周学习计划
"昨天任务没做完"       - 触发任务欠账处理
"我是夜型人"          - 设置为夜型人模式
"感觉有点累"          - 调整计划强度
"解析课表"             - 解析上传的课表文件
```

## 触发条件

### 强触发（直接调用）
- "生成考研计划"
- "今日学习计划"
- "考研时间表"
- "学习安排"（考研语境）
- "任务没做完" / "欠账"（触发补丁1）

### 自动触发
- 周日晚上（触发补丁2：周日复盘）
- 时段不满足数学最小时长（触发补丁3）

### 弱触发（询问确认）
- "解析课表"
- "周计划"（需确认考研语境）
