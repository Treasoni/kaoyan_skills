# kaoyan-info

> 考研信息收集与文档管理 AgentSkill

## 简介

`kaoyan-info` 是一个专注于收集中国研究生入学考试（考研）最新信息，以及创建和管理考研相关文档的 AgentSkill。与 `kaoyan-math` 技能不同，本技能关注**信息收集**和**文档管理**，而非具体的数学学科内容生成。

## 功能特性

### 信息收集

- **院校招生信息**：招生简章、专业目录、招生人数
- **分数线数据**：历年复试线、国家线、自划线院校分数线
- **专业目录变化**：新增/撤销专业、考试科目调整
- **参考书目信息**：初试/复试参考教材、版本要求
- **复试政策**：复试办法、调剂政策、录取规则

### 文档管理

- 创建院校考研指南
- 创建专业课笔记框架
- 创建复习计划表
- 创建资料清单
- 更新现有文档数据

## 使用方法

### 触发条件

使用以下任一方式触发此技能：

- "收集考研信息"、"收集XX大学考研信息"
- "创建考研指南"、"生成考研文档"
- "更新分数线"、"更新招生信息"
- "查找XX大学XX专业"信息
- "XX学校XX专业招多少人"、"XX专业分数线"

### 执行模式

本技能采用分级执行模式，确保对文档操作有完全控制权：

| 模式 | 行为 | 文件修改 | 适用场景 |
|------|------|----------|----------|
| 仅收集信息 | 收集并展示信息 | 不创建任何文件 | 只想了解信息，不需要保存 |
| 生成文档草稿 | 生成内容预览 | 不保存文件 | 想预览文档内容后再决定 |
| 更新现有文档 | 修改已存在文档 | 需用户确认 | 明确要求更新现有文档 |

## 配置

首次使用时，需要配置考研文档保存目录。
