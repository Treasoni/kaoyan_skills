# 错题精华提炼

## 触发条件

触发表达：
- "提炼XX的错题精华"
- "为XX生成精华本"
- "提取错题中的精华"
- "错题精华总结"
- "帮我整理错题中的做题结论"
- "从错题中提取解题思路"

**适用科目**：考研数学、线性代数、数字电子技术

---

## 核心功能

1. **扫描错题本** — 读取指定模块的 `错题本.md`，逐题解析
2. **提取7类精华** — 核心口诀、避坑要点、关键认知、解题方法、检查清单、核心陷阱、错误模式
3. **灵活格式适配** — 自动识别不同科目/模块的格式差异（高数/线代/数电）
4. **生成提炼笔记** — 在错题本同级目录生成 `💡错题精华提炼.md`
5. **保留 LaTeX** — 所有数学公式原样保留
6. **建立回链** — 每条精华都带 `[[错题本#锚点]]` 链接回原错题
7. **覆盖更新** — 重复运行时全量重新生成，避免内容重复

---

## 工作流程

```
[用户指定模块]
      ↓
[1. 解析模块名 → 映射到错题本路径]
      ↓
[2. 读取错题本文件]
  ├─ 文件不存在 → 报错，列出可选模块
  └─ 文件存在 → 继续
      ↓
[3. 逐题扫描解析 (Phase 1: 数据提取)]
  对每个 ## 错题NN：标题 块：
  ├─ 提取索引表的核心陷阱
  ├─ 提取 [!tip] / 💡 块中的口诀、避坑要点、清单、认知
  ├─ 提取 ✅ 正确解法中的方法名和步骤
  ├─ 提取 [!personal] / ❌ 块中的致命伤、思维误区
  └─ 记录错题序号和标题（用于回链）
      ↓
[4. 分类聚合去重 (Phase 2: 整理)]
  ├─ 按7个类别分组缓存
  ├─ 同口诀出现在多题 → 合并记录所有来源
  └─ 统计各类别提取数量
      ↓
[5. 生成提炼笔记 (Phase 3: 输出)]
  ├─ 组装按类组织的内容
  ├─ 生成 wikilink 回链
  └─ 写入 💡错题精华提炼.md
      ↓
[6. 输出统计摘要]
  - 显示提取了多少条精华
  - 显示文件路径
```

## 与其它技能的协同

| 技能 | 关系 |
|------|------|
| `mistake-book` | 错题本的数据来源，提取前确保错题本已整理 |
| `kaoyan-math` / `kaoyan-electronics` | 获取模块知识图谱结构 |
| `kaoyan-plan` | 提炼完成后可更新学习进度 |
| `understanding` | 验证失败时可建议整理错题后再提炼 |

---

## 注意事项

- 提炼笔记**覆盖式更新**（不追加），确保与错题本同步
- `[!personal]` 块内容仅提取错误模式，不复制个人笔记原文
- 数字电子技术错题本可能有图片嵌入，图片不提取但标注"参见原错题配图"
- 如果某模块无错题记录，提示用户而非生成空文件
