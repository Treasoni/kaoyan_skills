## 锚点生成规则（复用 mistake-book）

错题标题 → 锚点转换：
- 去掉所有标点符号（`：` `"` `（` `）` 等）
- 空格替换为 `-`
- LaTeX 公式中的 `$` 符号不参与锚点生成（已被移除）
- 例：`错题01：泰勒展开精度陷阱` → `#错题01泰勒展开精度陷阱`
- 例：`错题02：方程中含自身极限的"套娃"问题` → `#错题02方程中含自身极限的套娃问题`
- 例：`错题05：遇到 $\ln(1 + \text{无穷大})$ 的处理套路` → `#错题05遇到-ln1-text无穷大的处理套路`

## Phase 1: 逐题解析 — 提取规则

每个错题条目以 `## 错题\d+：` 开头，以下一个 `## 错题\d+：` 或文件末尾为边界。

### 1.1 提取错题元信息

从标题行和索引表提取：

```
错题序号: NN（两位数）
错题标题: 标题文本（去除"错题NN："前缀）
锚点链接: #错题NN{锚点化标题}
```

### 1.2 从索引表提取核心陷阱

索引表位于文件开头，在 `> [!info]` 块内。每行格式：

```
| [NN](#错题NN{锚点}) | {标题} | {核心陷阱描述} | {难度} | ... |
```

提取每行的"核心陷阱"列内容，与错题序号关联。

### 1.3 从 `[!tip]` / `💡` 块提取（主要提取源）

#### A. 核心口诀

匹配规则（按优先级）：
1. `**核心口诀**："{口诀}"` → 提取引号内文本
2. `**核心口诀**：{口诀}` → 提取冒号后文本
3. `**核心套路总结**` 下的表格/列表 → 提取关键步骤
4. `**核心思路**：` → 提取冒号后文本
5. 任何包含 `口诀` 的粗体行

#### B. 避坑要点

匹配规则（按优先级）：
1. `**避坑要点**：` 下的编号列表 → 逐条提取
2. `**避坑指南**：` 下的内容 → 提取要点
3. `**易错点提醒**：` → 提取提醒内容
4. `**关键动作**：` 下的编号列表 → 提取
5. 任何包含 `避坑\|易错` 的粗体块

#### C. 关键认知

匹配规则（按优先级）：
1. `**关键认知**：` 下的内容
2. `**关键结论**：` 或 `**关键结论与避坑指南**：` 下的内容
3. `**盲区纠正**：` → 提取纠正后的正确认知
4. `**认知纠正**：` → 提取
5. `**核心原则**：` → 提取
6. `**复盘总结**：` → 提取总结性内容
7. 任何包含 `认知\|纠正\|原则\|复盘` 的粗体块

#### D. 第一反应检查清单

匹配规则：
1. `**第一反应检查清单**：` 下的内容
2. `**下次遇到这类题目，{动作}**` → 提取动作描述
3. 包含"清单"或"第一时间"的行

### 1.4 从 `✅ **正确解法**` 提取解题方法

```
✅ **正确解法**（{方法名称}）：
```

- 提取方法名称（括号内文本）
- 提取步骤列表（`**第X步：{步骤名}**`）
- 如果没有明确的方法名，从标题或第一个步骤推断
- 如果解法中包含 `**核心套路总结**` 表格（如错题02的四步法表格），提取整表

### 1.5 从 `[!personal]` / `❌` 块提取错误模式

匹配规则：
1. `**致命伤**：` → 作为错误模式的核心描述
2. `> | 困惑点 | 表现 |` 表格 → 提取困惑点和表现
3. `**错误原因**：` → 提取
4. `**典型错误轨迹**` → 提取错误路径描述
5. 线代格式 `❌ **我的错解与误区**` → 提取错误做法描述

### 1.6 LaTeX 保留规则

- 所有 `$...$` 内联公式原样保留
- 所有 `$$...$$` 显示公式原样保留
- 提取时不要截断公式（确保 `$` 配对完整）
- 表格内如有 `|` 符号（如行列式、分段函数），替换为 `\vert` 或将内容移出表格

### 2.2 去重规则

- 核心口诀：完全相同（忽略标点差异）的合并，记录多个来源
- 避坑要点：语义相似（同一陷阱名）的合并
- 其它类别：保留所有条目，不合并（即使相似）

### 2.3 排序规则

- 核心口诀：按来源错题序号排序
- 避坑要点：按来源错题序号排序
- 关键认知：按来源错题序号排序
- 解题方法：按来源错题序号排序
- 核心陷阱：按难度星级降序
- 错误模式：按来源错题序号排序
- 检查清单：保持逻辑顺序

## 特殊格式处理

### 表格中 LaTeX 管道符

当提取内容需要放入 markdown 表格且包含 `|` 时（如分段函数、行列式），使用以下策略：

1. 绝对值和范数：用 `$\lvert ... \rvert$` 替代 `$|...|$`
2. 分段函数：用 `$\begin{cases} ... \end{cases}$`（无 `|`）
3. 行列式：用文字描述替代，如"行列式展开式"
4. 如果需要保留矩阵/行列式格式，将该项移出表格，改为独立公式

### 公式中的 `\text{}` 命令

`$\ln(1 + \text{无穷大})$` 这类在锚点生成时 `\text{}` 及其内容被去掉，结果可能不理想。但回链锚点仍然有效（Obsidian 的模糊匹配）。

### 图片引用

如果错题本中有 `![[assets/错题本/xxx.png]]`，在提炼笔记中不嵌入图片，但可在相关内容旁标注：

```markdown
*（参见原错题中的配图）*
```
