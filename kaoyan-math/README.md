# 🧠 Kaoyan Math System v3.0.0
考研数学结构化学习与迭代笔记系统

---

## 📌 项目定位

Kaoyan Math System 是一个基于 Claude Code 构建的 **考研数学学习系统化方案**。

**v3.0.0 新增特性**：
- ✅ **LaTeX格式强制** - 所有数学公式自动转换为LaTeX，确保Obsidian正确渲染
- ✅ **知识点主动关联** - AI主动提示跨章节关联（如洛必达法则与变限积分求导的结合）
- ✅ **MemOS持久化集成** - 真正的"千人千面"个性化笔记，记住所有错题历史

本项目的目标不是简单生成笔记，而是：

> 构建一个可持续迭代的数学学习系统
> 让"理解 → 反馈 → 修正 → 强化"成为循环

适用于：

- 考研数学（数一 / 数二 / 数三）
- 使用 Obsidian 进行知识管理
- 希望系统化学习的考研学生
- 需要跨设备同步学习进度的用户

# 🚀 使用示例

## 生成笔记（含LaTeX格式）

```
这是我的极限笔记：
洛必达法则：lim f(x)/g(x) = lim f'(x)/g'(x)，当f(x)和g(x)都趋向0或无穷大时。

帮我生成考研数一的极限学习笔记。
```

**AI输出**（v3.0自动应用LaTeX格式）：
```markdown
## 原始定义
$$
\lim_{x \to x_0} \frac{f(x)}{g(x)} = \lim_{x \to x_0} \frac{f'(x)}{g'(x)}
$$

## 定理条件 ⚠️
1. $\lim f(x) = \lim g(x) = 0$ 或 $\infty$（必须是 $0/0$ 或 $\infty/\infty$ 型）
2. $f(x)$ 和 $g(x)$ 在某去心邻域内可导，且 $g'(x) \neq 0$
3. $\lim f'(x)/g'(x)$ 存在或为 $\infty$
```

## 更新笔记

```
我对洛必达法则的条件不理解，为什么要求g'(x)≠0？
```

## 个性化提醒（基于MemOS）

```
复习洛必达法则时，系统自动提示：
⚠️ 你在"条件遗漏"方面已出错3次，特别注意验证0/0或∞/∞型条件
```

## 知识点主动关联

```
当学习"洛必达法则"时，系统自动提示：
- 注意：当遇到变限积分求导时，通常会结合洛必达法则考查
- 建议：同时复习 [[定积分应用]] 中的变限积分部分
```
