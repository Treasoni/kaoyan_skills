# 🧠 Kaoyan Math System  
考研数学结构化学习与迭代笔记系统

---

## 📌 项目定位

Kaoyan Math System 是一个基于 Claude Code 构建的 **考研数学学习系统化方案**。

本项目的目标不是简单生成笔记，而是：

> 构建一个可持续迭代的数学学习系统  
> 让“理解 → 反馈 → 修正 → 强化”成为循环

适用于：

- 考研数学（数一 / 数二 / 数三）
- 使用 Obsidian 进行知识管理
- 希望系统化学习的考研学生

---

# 🎯 核心理念

传统学习流程：

听课 → 做题 → 犯错 → 再犯错

本系统学习流程：

原始笔记  
↓  
结构化生成  
↓  
学习理解  
↓  
问题反馈  
↓  
笔记更新  
↓  
错误模式沉淀  
↓  
强化训练  

它不是一次性整理工具，而是一个可迭代的数学认知系统。

# ⚠ 使用边界

本系统提供：

- 结构支持
- 内容整理
- 反馈修正
- 迭代优化

但不替代：

- 做题量
- 推导训练
- 计算熟练度
- 真题反复刷

> 系统是加速器  
> 不是发动机
